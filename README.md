# 📘 Segundo Cérebro de Warren Buffett

## 🎯 Objetivo
Este repositório tem como propósito **criar um segundo cérebro** inspirado nos ensinamentos de **Warren Buffett**, aplicando suas metodologias de investimento e filosofia de vida.  
A ideia é organizar, sintetizar e aplicar os princípios de **Value Investing**, disciplina financeira e mentalidade de longo prazo, transformando-os em conhecimento prático e acessível.

---

## 🧠 O que é um "Segundo Cérebro"?
Um **segundo cérebro** é um sistema de organização pessoal que permite armazenar, conectar e recuperar informações de forma eficiente.  
Aqui, o foco é construir esse sistema baseado nos **ensinamentos de Buffett**, para que investidores e estudantes possam aprender e aplicar seus métodos de forma estruturada.
